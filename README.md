# Billing History Management System 

A simple console-based Java application to manage billing records using a Singly Linked List.

## Features
- Add billing records  
- View billing history  
- Delete records by Bill ID  

## Technologies
- Java  
- Linked List  
- Scanner (user input)

## Project Structure

BillingApp.java
│
├── BillingRecordd // Node class for linked list
├── BillingHistory // Linked list operations
└── BillingApp // Main class (Menu-driven program)

## How to Run the Project

1. Make sure Java (JDK) is installed on your system  
2. Clone the repository: git clone https://github.com/Kashish1530/Billing-Management-App.git
3. Compile the program: javac BillingApp.java
4. Run the program: java BillingApp

## Menu Options

1. Add Billing Record
2. View Billing History
3. Delete Billing Record
4. Exit
